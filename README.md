# Budget-Tracker
a simple app to track your daily income and expenses

## PROBLEM:
People often find it difficult to track their daily income and expenses and this leads to a lack of financial awareness.

## SOLUTION:
Building a simple web app that allows users to record and view their transactions

## MVP FEATURES:
- User login
- Dashboard showing balance
- Add transactions(income or expense)
- View of transactions

## Tech stack:
Frontend: HTML, CSS, Javascript

Backend: Node.js

Database: MongoDB
